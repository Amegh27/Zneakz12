 ZNEAKZ

A premium shoe e-commerce website built with Node.js, EJS, and MongoDB.

 About the Project

Zneakz is a modern and fully functional footwear e-commerce platform designed to deliver a smooth and intuitive shopping experience.
Built with performance and simplicity in mind, Zneakz includes complete user authentication, admin controls, order management, wallet features, offers, and more.

 Tech Stack

Frontend: EJS
Backend: Node.js, Express
Database: MongoDB

 Key Features
 User Features

Secure user authentication

Google login (Google OAuth)

OTP verification for signup & password reset

Clean, minimal, and responsive UI

Wallet system for easy refunds & payments

Refer & Earn reward mechanism

Wishlist & cart management

Order tracking

Shoe size & variant selection

 Admin Features

Complete admin dashboard

Manage users, products, categories, and shoe variants

Track and update orders

Analytics & sales reports

Download reports as PDF

Manage coupons, discounts & offers

Banner and homepage content management
